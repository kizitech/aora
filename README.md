# Aora

Aora is a modern mobile app UI implementation built from a Figma community design, focused on a clean onboarding experience and a polished app interface.

## About

This project is a frontend implementation of the Aora app design. It focuses on translating the Figma UI into a responsive, clean, and production-ready mobile interface.

## Features

- Clean and modern mobile UI
- Responsive layout
- Smooth visual hierarchy
- Reusable components
- Pixel-aware implementation from Figma

## Tech Stack

- React Native
- Expo
- TypeScript
- Tailwind CSS / NativeWind
- Figma
